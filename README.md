# 2D Graphics Editor

A menu-driven console application in C that draws 2D shapes on a character array.

## Features
- Draw rectangles, circles, triangles, and lines using `*`
- Delete selected shapes
- Display the canvas

## How to Run
```bash
gcc "graphics_editor.c" -o editor
./editor
```

## Concepts Used
- Arrays and functions in C
- Loops and conditionals
- Basic 2D graphics logic
